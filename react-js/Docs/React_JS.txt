React Components::
=================
-Component is template with pre-defined design, styles and functionality 
-design is create using HTML 
- Styles are configure using CSS 
- functionality is defined by using JavaScript/TypeScript 
- React uses javaScript extenstion known as JSX
- React component can be created by using 2 techniques 
 a) JavaScript class 
 b) JavaScript Function

Function Component Rules:
1) A component function cant be void
2) ITs must return JSX element 
3) Its must start with Uppercase letter 
4) Component JSX must return only one fragment 
5) Every element in component must have an end token
